## Storage Structure

### Backend Data Structure

```javascript
// Participants (username is the key!)
participants = {
  "Alice": { "name": "Alice" },
  "Bob": { "name": "Bob" },
  "Charlie": { "name": "Charlie" }
}

// Answers (tuple of username + question_id as key)
answers = {
  ("Alice", 1): 1,      // Alice answered option 1 for question 1
  ("Alice", 2): 0,      // Alice answered option 0 for question 2
  ("Bob", 1): 2,        // Bob answered option 2 for question 1
  ("Charlie", 1): 1,    // Charlie answered option 1 for question 1
  ("Charlie", 2): 3     // Charlie answered option 3 for question 2
}

// Scoreboard calculation
for each username in participants:
  correct = 0
  for each question with correct answer:
    if answers[(username, question.id)] == question.correct_index:
      correct += 1
```
